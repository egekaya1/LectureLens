# LectureLens
LectureLens allows students of all kinds to input course material and receive a planned study program to achieve their goals. Features include summaries, flashcards, Q&amp;A's, schedules and more. Built with TypeScript, Next.js, Supabase as tech stack.

## Local setup
1. copy `.env.example` -> `.env.local` and fill values
2. yarn install
3. yarn dev

## Tech
- Next.js + TypeScript
- TailwindCSS
- Supabase (Auth, Postgres, Storage)
